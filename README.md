## TECH STACK

- Stack : `NextJS`
- UI : `TailwindCSS and ShadcnUI`
- Buttons : `Syntax UI`
- Images/svg : `Popsy`
- Icons : `Luicde`
- Hosting : `Vercel`

---

## Setup

1. Fork and Clone the repo
2. If you don't have pnpm run `npm install -g pnpm`
3. run `pnpm install`
4. run `pnpm run dev`
5. Create, a new Branch, make code change make a PR (Contributions are welcome)
6. HAPPY CODING!
